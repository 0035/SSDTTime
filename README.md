SSDTTime
==========
A simple tool designed to make creating SSDTs simple.

### Supported SSDTs:
- SSDT-EC
    - OS-aware fake EC
- SSDT-PLUG
    - Sets plugin-type = 1 on CPU0/PR00
- SSDT-HPET
    - Patches out IRQ conflicts
